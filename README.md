I am an Assistant Professor of Physics at Brock University in St. Catharines, Ontario, Canada. My research focuses on causality in general relativity and quantum mechanics, and scientific computing.

In numerical computation, I am interested in writing high-performance software for scientific research using optimized algorithms, multithreading, GPU programming, and other cutting-edge techniques. I am currently working on C++ code for high-quality visualization of arbitrary spacetime metrics using relativistic ray tracing, for use in my own research and by other general relativity researchers.

In symbolic computation, I am interested in implementing advanced concepts in mathematics and physics using computer algebra systems. I am currently working on an object-oriented tensor toolkit, intended to streamline and simplify symbolic calculations in general relativity and differential geometry, with implementations in both Mathematica and Python.

I am also interested in making scientific computing more user-friendly and accessible to both novice students and established researchers. I write toolkits, code libraries, and other reusable code, with special focus on producing clear, lightweight, thoroughly-documented, and easy-to-use code, which is made freely and publicly available on my GitHub page. Examples include my <a href="https://github.com/bshoshany/thread-pool">C++ thread pool class</a> and my <a href="https://github.com/bshoshany/multithreaded-matrix">multithreaded C++ matrix class template</a>.

Please see [my personal website](http://baraksh.com/) for more information.
